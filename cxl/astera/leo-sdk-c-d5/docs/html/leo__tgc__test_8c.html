Detailed Description

api test demonstrates usage of Leo SDK TGC test API calls. This is recommended for:

  • Print SDK version
  • Iinitialize Leo device
  • Run TGC test example sudo ./leo_tgc_test 0 0xaabbccdd 1 1 1
    Parameters
    [1-in]start address, default 0
    [2-in]dataPattern, default value 0xaabbccdd
    [3-in]Transaction count[0 to 15], 0 for 256 ... and 15 for 32G
    [4-in]Write enable 0-disable 1-Enable
    [5-in]Read enable 0-disable 1-Enable
    For more details on the args supported & usage sudo ./leo_tgc_test -help
