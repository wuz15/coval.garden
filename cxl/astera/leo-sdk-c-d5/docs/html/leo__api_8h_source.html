435/*
436 * @brief Fuction to throttle the bandwidth of DDR based on temperature
437 * threshold.
438 * Threshold setting will change the BW available to the required percentage
439 * of the total BW.
440 * For example, Full is 100% of available BW,
441 * 75 is 75% of available BW,
442 * 50 is 50% of available BW,
443 * 25 is 25% of available BW.
444 * 10 is 12.5% of available BW.
445 *
446 * @param[in] leoDevice - Leo device structure
447 * @param[in] leoHandle - Leo device handle
448 * @param[in] Max - is Full BW (100->64)
449 * @param[in] count - is throttling percentage
450 * (25->16, 50->32, 75-> or 100->64)
451 */
