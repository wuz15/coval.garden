\doxysubsection{Detailed Description}
Struct defining detailed Pseudo Port status, including electrical parameters. 

Note that each Leo Link has two Pseudo Ports, and Upstream Pseudo Port (USPP) and a Downstream Pseudo Port (DSPP).
